## 5.1 if-else 结构
* 注意 condition 并不需要括号

* 关键字 `if` 和 `else` 之后的左大括号 `{` __必须__ 和关键字在同一行

```go
if condition {
    // do something 
}

if condition {
    // do something 
} else {
    // do something 
}

if condition1 {
    // do something 
} else if condition2 {
    // do something else    
} else {
    // catch-all or default
}
```
